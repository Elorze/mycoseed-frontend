# MycoSeed Pixel Art Setup

To enable the 8-bit visualization, you must install D3.js.

Please run the following command in your terminal:

```bash
# If using npm
npm install d3 @types/d3

# If using yarn
yarn add d3 @types/d3
```

## Features Added

1.  **Pixel UI Components**:
    *   `components/pixel/PixelButton.vue`
    *   `components/pixel/PixelCard.vue`
    *   `components/pixel/PixelAvatar.vue`

2.  **Layout**:
    *   Mario-style header and footer.
    *   Sky blue background with cloud decorations.

3.  **Graph**:
    *   Force-directed graph using D3 (requires installation).
    *   Located in `components/graph/NetworkCanvas.vue`.
